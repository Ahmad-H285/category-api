#Installation

- Clone the repository
- Run `composer install`
- Run `php artisan serve`
- Run the migrations by using `php artisan migrate`
- Run the created seeders `php artisan db:seed`
- Use this sample api and change the parameters value to see the results:
127.0.0.1:8000/api/v1/ads?category=Shoes&tag=t-shirts

Thats it, you are all set
